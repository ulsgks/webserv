500 - Server Fucked Up

Something went wrong on our end. The server shit the bed.

This is a custom error page for when the server encounters some internal fuckery.

It could be a 500 error as a 503 (Service Unavailable) or a 504 (Gateway Timeout).

In the config file it looks like this:

error_page 500 503 504 /error/50x.html;
